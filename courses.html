Course Catalogue

Explore the full Computer Science curriculum — from introductory programming to advanced research topics

Year 1

First Year Modules

CS113 Semester 1 16 credits

Computer Science for Actuarial Studies

An introduction to computer science tailored for actuarial science students. Covers financial programming concepts, spreadsheet automation with Excel/VBA, and fundamental programming constructs.

Prerequisites: None

Financial Programming Spreadsheets VBA Introductory Algorithms
CS114 Semester 1 16 credits

Introductory Computer Science 1

The foundation module for all Computer Science students. Introduces programming in Python/Java, covering basic control flow, data representation, arrays, recursion, and an introduction to object-oriented programming.

Prerequisites: NSC Mathematics

Python/Java Control Flow Data Representation Arrays Recursion OOP Intro
CS144 Semester 2 16 credits

Introductory Computer Science 2

Continues from CS114. Covers fundamental data structures (linked lists, trees, graphs), classical sorting and searching algorithms, full object-oriented programming with design patterns, and an introduction to computational complexity theory.

Prerequisites: CS114

Data Structures Algorithms OOP Design Patterns Big-O Complexity
Year 2

Second Year Modules

CSE214 Semester 1 16 credits

Object-Oriented Programming

Advanced object-oriented programming principles. In-depth study of design patterns (GoF), SOLID principles, refactoring techniques, unit testing, and software quality. Uses Java.

Prerequisites: CS144

Design Patterns SOLID Principles Refactoring Unit Testing Java
CS214 Semester 1 16 credits

Data Structures and Algorithms

Classical data structures and algorithm design. Covers advanced trees (AVL, red-black), heaps, hash tables, graph algorithms (Dijkstra, Floyd-Warshall, Kruskal), dynamic programming, and algorithm analysis.

Prerequisites: CS144

AVL Trees Hash Tables Graph Algorithms Dynamic Programming Algorithm Analysis
CS244 Semester 2 16 credits

Computer Architecture

Computer organization and architecture from first principles. Assembly language programming, memory hierarchy and caching, processor design, pipelining, and introduction to operating systems concepts.

Prerequisites: CS144

Assembly Language Memory Architecture CPU Design Pipelining OS Concepts
SC272 Both Semesters 16 credits

Scientific Computing

Practical computing skills for science students. Linux command line, shell scripting, Python for scientific computing (NumPy, SciPy, Matplotlib), numerical methods, and version control with Git.

Prerequisites: CS114

Linux Shell Scripting Python NumPy Matplotlib Git Numerical Methods
Year 3

Third Year Modules

CS313 Semester 1 16 credits

Computer Networks

Principles and protocols of computer networks. The TCP/IP stack in depth — physical, data link, network, transport, and application layers. Socket programming, network security basics, and performance analysis.

Prerequisites: CS244

TCP/IP Socket Programming Network Protocols Security Performance
CS314 Semester 1/2 16 credits

Concurrency

Concurrent and parallel programming. Thread management, synchronization primitives (mutexes, semaphores, monitors), deadlock detection and prevention, parallel algorithms, and introduction to distributed systems.

Prerequisites: CS244

Threads Synchronization Deadlock Parallel Algorithms Distributed Systems
Honours

Honours Modules

CSc771 16 credits

Advanced Algorithms

Lecturer: Willem Bester

Advanced topics in algorithm design and complexity theory. NP-completeness proofs, approximation algorithms for hard problems, randomized algorithms, streaming algorithms, and advanced data structures.

Prerequisites: CS Honours student

NP-Completeness Approximation Algorithms Randomized Algorithms Complexity Theory
CSc772 16 credits

Advanced Topics I: Vulnerability Discovery and Exploitation

Lecturer: Fabian Yamaguchi

Modern vulnerability research techniques. Static and dynamic analysis, fuzzing (grey-box, coverage-guided), symbolic execution, binary analysis, CVE research methodology, and responsible disclosure.

Fuzzing Static Analysis Symbolic Execution Binary Analysis CVE Research
CSc773 16 credits

Advanced Topics II: Ontology Engineering

Lecturer: Maria Keet

Theoretical foundations and practical construction of ontologies. OWL 2 language, SPARQL queries, ontology design patterns, reasoning with description logics, and Semantic Web applications.

OWL 2 SPARQL Ontology Design Description Logic Semantic Web
CSc781 32 credits

Honours Project

An independent research project under faculty supervision. Students investigate a novel computer science problem, conduct a literature review, perform experiments or develop a system, and write a research-quality thesis.

Prerequisites: CS Honours student

Research Methods Literature Review Thesis Writing Oral Defense
